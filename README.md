# EMAIL-SORTER

This app can receive a list of emails which you can search and mass delete to clean your inbox.

## Requirements

- PHP 8.2
- Composer
- php-mbstring
- php-imap

## Getting Started

```bash
git clone git@github.com:niwee-productions/email-sorter.git
cd email-sorter
./startup.sh
```

The app will run at `http://localhost:8000`
